## Dark Mode Support

All components automatically adapt to dark mode:

- Background colors use dark variants
- Text colors adjust for contrast
- Border colors use dark-mode-friendly grays
- Status colors maintain visibility

### Dark Mode Colors

- Background: gray-800 / gray-900

- Borders: gray-700
- Active segment: teal-950/30 background
